Programming Fundamentals

  • Variables & Data Types: int, float, string, boolean; constants; variable naming rules
  • Operators: Arithmetic (+, -, *, /, %), comparison (==, !=, >, <, >=, <=), logical (&&, ||, !)
  • Control Flow: if, else, elif, switch/case, ternary operator
  • Loops: for, while, do-while; break & continue
  • Functions: Parameters, return values, recursion, scope (local/global)
  • Input & Output: Reading user input, printing output, string formatting
  • Worked Example: 
    // Python
for i in range(1, 6):
    print(i * 2)

// Output: 2 4 6 8 10
  • Tips: Comment code, write reusable functions, meaningful variable names, small modular code

Data Structures

  • Arrays / Lists: Indexed collections, fixed vs dynamic, insert/delete/access
  • Stacks: LIFO, push/pop, examples: undo, browser history
  • Queues: FIFO, enqueue/dequeue, examples: print queues, task scheduling
  • Linked Lists: Singly & doubly linked; nodes = data + pointer; traversal
  • Dictionaries / Hash Maps: Key-value storage, lookups, hash functions
  • Trees: Binary trees, BST, traversal methods (preorder, inorder, postorder)
  • Graphs: Nodes & edges, adjacency list/matrix, BFS & DFS basics
  • Worked Example - Stack: 
    Stack operations:
Push 10 → [10]
Push 20 → [10, 20]
Pop → 20, stack: [10]
Pop → 10, stack: []
  • Tips: Draw diagrams, visualise operations, practise common examples

Algorithms

  • Sorting: Bubble, Selection, Insertion, Merge, Quick; step-by-step examples
  • Searching: Linear vs Binary search (on sorted arrays)
  • Recursion: Base case, recursive call, stack trace examples (factorial, Fibonacci)
  • Trace Table Example - Fibonacci(n=5): 
    Call stack:
fib(5) → fib(4)+fib(3)
fib(4) → fib(3)+fib(2)
...
Result: 0,1,1,2,3,5
  • Big O / Complexity: Time & space, best/average/worst case
  • Tips: Trace with small input, write pseudocode first, understand logic

Logic & Boolean

  • Boolean Algebra: AND, OR, NOT, XOR; truth tables
  • Logic Gates: AND, OR, NOT, NAND, NOR, XOR, XNOR; truth table examples
  • Conditional Statements: Apply boolean logic in code
  • Flowcharts & Pseudocode: Represent algorithms visually, plan logic
  • Worked Example: 
    Input: a=True, b=False
Output: a AND b → False
a OR b → True
NOT a → False
  • Tips: Draw truth tables, simplify expressions, check all cases

Data Representation

  • Binary Numbers: Base-2, 0 & 1, used internally by computers
  • Decimal & Hexadecimal: Base-10 (human), Base-16 (short form for binary)
  • Converting Bases: Binary ↔ Decimal ↔ Hexadecimal; examples included
  • Binary Arithmetic: Addition, subtraction, overflow example 
    0101 + 0011 = 1000 (5+3=8)
  • Two’s Complement: Represent negative numbers, example: -5 → 11111011
  • Character Encoding: ASCII, Unicode (UTF-8), mapping characters to binary
  • Floating Point Representation: IEEE 754, mantissa & exponent
  • Data Size: Bits, bytes, KB, MB, GB, TB
  • Worked Example: 
    // Binary to Decimal
1011 → 1*8 + 0*4 + 1*2 + 1*1 = 11
  • Tips: Memorise base conversions, practise binary arithmetic, understand data limits

Hardware & Computer Basics

  • Computer Components: CPU, RAM, storage, input/output devices
  • Memory & Storage: Primary vs secondary, cache, registers
  • Boolean Logic in Hardware: Logic gates, circuits, digital vs analog signals
  • Networking Basics: IP addresses, protocols, LAN/WAN, client-server
  • Tips: Memorise key components, visualise circuits, understand data flow

Exam & Coding Tips

  • Write pseudocode first
  • Trace code with sample input
  • Check edge cases (empty arrays, zero values, negative numbers)
  • Comment tricky sections for clarity
  • Time yourself on practice coding problems
  • Memorise common algorithms & data structures
  • Use consistent variable names & indentation
  • Plan before implementing complex logic (loops/recursion)
  • Draw diagrams for data structures and logic
