Algebra

  • Expressions: Expand brackets, factorise, simplify like terms, difference of squares, cubic factorising
  • Linear Equations: Solve 2-step and multi-step equations, fractions, decimals
  • Simultaneous Equations: Solve algebraically (substitution, elimination) and graphically
  • Quadratic Equations: Factorisation, completing the square, quadratic formula
    • Quadratic formula: x = [-b ± √(b²−4ac)] / 2a
  • Worked Examples:
    • 2x + 5 = 13 → 2x = 8 → x = 4
    • Simultaneous: x + y = 7, x − y = 1 → x = 4, y = 3
    • Quadratic: x² − 5x + 6 = 0 → (x−2)(x−3) = 0 → x = 2 or 3
  • Tips: Always show working, check solutions by substitution, write formulas first
  • Mini Table – Key Algebra Rules:
    RuleExample
    a² − b² = (a−b)(a+b)x² − 9 = (x−3)(x+3)
    (a+b)² = a² + 2ab + b²(x+4)² = x² + 8x + 16
    (a−b)² = a² − 2ab + b²(x−2)² = x² − 4x + 4

Geometry

  • Angles: Parallel lines (alternate, corresponding, co-interior), triangle sum (180°), polygon sum ((n−2)×180°), angles at a point (360°)
  • Shapes & Properties: Circle, rectangle, square, triangle, trapezium, parallelogram, rhombus, prism, pyramid
  • Area & Perimeter:
    • Rectangle: A = l×w, P = 2(l+w)
    • Square: A = s², P = 4s
    • Triangle: A = ½×base×height
    • Circle: A = πr², C = 2πr
    • Trapezium: A = ½(a+b)h
    • Parallelogram: A = base×height
  • Volume & Surface Area:
    • Cuboid: V = l×w×h, SA = 2(lw + lh + wh)
    • Cube: V = s³, SA = 6s²
    • Cylinder: V = πr²h, SA = 2πr(h+r)
    • Sphere: V = 4/3πr³, SA = 4πr²
    • Pyramid: V = 1/3 × base area × height
    • Cone: V = 1/3 × πr²h, SA = πr(l + r)
  • Worked Examples:
    • Triangle: base=5, height=8 → A=20
    • Cylinder: r=3, h=10 → V ≈ 282.7
  • Tips: Draw diagrams, label sides & angles, memorise formulas, always check units

Trigonometry

  • Basic Ratios: sinθ = opposite/hypotenuse, cosθ = adjacent/hypotenuse, tanθ = opposite/adjacent
  • Pythagoras: a² + b² = c², right triangles only
  • Worked Examples:
    • Triangle sides: 3 & 4 → hypotenuse = 5
    • sinθ = opposite/hypotenuse → sin30° = 1/2
    • tanθ = opposite/adjacent → tan45° = 1
  • Tips: Label triangles clearly, mnemonic “SOHCAHTOA”, practise with calculator for non-standard angles

Statistics & Probability

  • Mean, Median, Mode: Mean = sum / n, Median = middle value, Mode = most frequent
  • Range: Max − Min
  • Probability: P(event) = favorable / total, P(A or B) = P(A)+P(B)−P(A and B)
  • Worked Examples:
    • Die: P(2 or 5) = 2/6 = 1/3
    • Data set: 2,4,4,6,8 → Mean=24/5=4.8, Median=4, Mode=4
  • Tips: List all outcomes, check probabilities ≤ 1, show all steps, double-check sums

Exam & Problem-Solving Tips

  • Always show full working; partial marks count
  • Write formulas first before substitution
  • Draw diagrams wherever possible
  • Label all sides, angles, units
  • Check answers, especially for square roots, decimals, percentages
  • Memorise key formulas and shortcuts
  • Practice timed questions to improve speed
